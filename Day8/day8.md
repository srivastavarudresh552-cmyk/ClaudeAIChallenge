# 🌍 Personal Environmental Health Analyzer

A modern web application that provides real-time environmental health insights by analyzing Air Quality Index (AQI) and Water Quality data. The platform helps users understand environmental conditions in their area and offers actionable health recommendations based on current environmental metrics.

---

## 📖 Overview

The Personal Environmental Health Analyzer is designed to empower users with accurate and real-time environmental information. By combining air quality monitoring, water quality analysis, and health-risk assessment, the application provides a comprehensive view of environmental conditions affecting daily life.

Whether you're planning outdoor activities, monitoring local pollution levels, or checking water safety, this tool delivers meaningful insights through an intuitive dashboard.

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/personal-environmental-health-analyzer.git
```

Navigate to the project directory:

```bash
cd personal-environmental-health-analyzer
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
