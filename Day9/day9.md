
# 🥗 NutriScope

NutriScope is a comprehensive nutrition analysis and dietary planning platform that helps users track food intake, monitor macro and micronutrient consumption, identify nutritional deficiencies, and receive personalized dietary recommendations.

## 🌟 Features

### 👤 User Profile & Personalized Targets
- Age, gender, height, weight input
- Activity level selection
- Dietary preference selection
- Goal-based nutrition planning:
  - Maintain Weight
  - Weight Loss
  - Weight Gain
- Automatic daily nutrition target calculation

### 🍽 Food Logging
- Search and add food items
- Quantity and unit-based food tracking
- Daily food intake history
- Real-time nutritional calculations
- Delete food entries

### 📊 Dashboard Analytics
- Daily calorie tracking
- Energy consumed vs target
- Remaining calorie calculation
- Daily goal completion percentage

### 📈 Macro Nutrient Monitoring
Track:
- Calories
- Protein
- Carbohydrates
- Fat
- Dietary Fiber

Visualized through:
- Progress Bars
- Donut Charts
- Comparative Nutrient Analysis

### 🧬 Micronutrient Tracking
Monitor essential nutrients including:

- Vitamin A
- Vitamin C
- Vitamin D
- Vitamin E
- Vitamin B12
- Folate
- Iron
- Calcium
- Magnesium
- Zinc
- Omega-3

### 🎯 Deficiency Detection
Automatically identifies nutritional deficiencies and highlights:
- Critical deficiencies
- Moderate deficiencies
- Nutrients requiring attention

### 🕸 Micronutrient Radar
Interactive radar chart displaying:
- Current nutrient intake
- Target nutrient levels
- Nutritional balance overview

### 🍱 Smart Meal Planning
Generate dietary recommendations based on:
- User profile
- Nutrition goals
- Current deficiencies
- Dietary preferences

### 💡 AI-Powered Nutrition Insights
Provides:
- Suggested food additions
- Smart food swaps
- Portion adjustments
- Nutrient optimization tips

Examples:
- Include milk and curd for calcium
- Add spinach and chana for iron
- Add flaxseeds for Omega-3
- Increase leafy greens for folate
