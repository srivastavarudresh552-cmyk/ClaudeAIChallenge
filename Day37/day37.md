# 🧭 Task Compass

> **Learn how work flows through real organizations.**

Task Compass is an interactive workflow simulation that teaches users
how responsibilities are assigned, how tasks move across teams, and how
cross-functional collaboration works inside a modern tech company.

## 🚀 Features

-   Scenario-based learning
-   Three progressive stages
-   Interactive drag-and-drop workflow builder
-   Performance analytics and personalized feedback
-   Modern dark UI
-   Offline simulation

## 📚 Stages

### Stage 1 --- Who Owns This?

Assign the single role that should own an incoming task.

### Stage 2 --- Task Routing

Arrange the correct workflow from task creation to resolution.

Example:

``` text
Customer Support
      ↓
Engineering Manager
      ↓
Backend Developer
      ↓
DevOps Engineer
```

### Stage 3 --- Collaboration Challenge

Choose up to four roles that should collaborate on larger
cross-functional problems.

## 📈 Debrief

After the simulation, users receive scores for:

-   Ownership
-   Delegation
-   Collaboration
-   Workflow Thinking

along with strengths and improvement suggestions.
