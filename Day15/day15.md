# 🔱 Vedic Astrology Reading App

<div align="center">


> **An AI-powered Vedic Astrology analysis that generates complete birth chart readings, dasha analysis, life forecasts, and personalized remedies — based on classical Parashara and Jaimini systems.**

## ✨ Features

- 🪐 **Complete Birth Chart Generation** — North Indian chart with all 9 planetary placements
- 🔯 **Yoga & Dosha Detection** — Budhaditya, Malavya, Ruchaka, Viparita Raja Yoga and more
- 📅 **Vimshottari Dasha Analysis** — Mahadasha, Antardasha, and Pratyantar dasha breakdown
- 🌠 **Nakshatra & Pada** — Moon nakshatra, lord, and pada-level interpretation
- 💼 **Career & Wealth Forecast** — Job vs business, foreign opportunities, wealth timeline by age
- 💑 **Relationship & Marriage Analysis** — Timing windows, spouse characteristics, compatibility factors
- 📊 **5-Year Forecast Table** — Year-by-year prediction across career, money, relationships, and health
- 💊 **Personalized Remedies** — Mantras, donations, gemstones (only when astrologically justified)
- 🌗 **Dark Mode Support** — Fully responsive, adapts to system theme

## 🚀 Getting Started

## 📋 Input Parameters

The prompt collects the following details to generate a reading:

```
Full Name          → Used for personalization
Gender             → Affects certain Jaimini calculations
Date of Birth      → DD/MM/YYYY
Birth Time         → HH:MM AM/PM (accuracy matters for Lagna)
Birth Time Accuracy → Exact / Approximate / Unknown
Place of Birth     → For latitude/longitude and timezone
Current City       → For transit analysis
Relationship Status → Context for relationship predictions
Profession         → Context for career analysis
Top 3 Concerns     → Prioritizes the reading focus
```

## 📐 Astrological Systems Used

| System | Application |
|--------|-------------|
| **Parashara** | Core house significations, yogas, functional benefics/malefics |
| **Vimshottari Dasha** | Mahadasha, Antardasha, Pratyantar periods |
| **Jaimini** | Chara karakas, atmakaraka identification |
| **Nakshatra System** | Moon nakshatra, pada, lord interpretation |
| **Transit Analysis** | Current Jupiter/Saturn transit impact |

## 📊 Sample Output Sections
