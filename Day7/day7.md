# Claude Usage Strategy Generator

A personalized AI consultant that helps users choose the best Claude model, effort settings, and workflow based on their goals, profession, and daily tasks.

---

## Overview

Many Claude users are unsure:

* Which model should they use?
* When should they use Haiku, Sonnet, or Opus?
* What effort level is appropriate?
* How can they maximize quality while minimizing cost and token usage?

This prompt acts as a **Claude AI Expert, Productivity Consultant, and AI Workflow Architect**, analyzing the user's profile and generating a customized Claude usage strategy.

---

## Features

### User Profiling

Collects information about:

1. Current situation

   * Student
   * Professional
   * Freelancer
   * Founder

2. Primary activities

   * Coding
   * Learning
   * Research
   * Content Creation
   * Marketing
   * Resume Building
   * Career Preparation
   * Business Planning

3. Claude usage frequency

   * Occasionally
   * Daily
   * Heavy User

4. Required outputs

   * Fast Answers
   * Learning Support
   * Coding Help
   * Deep Research
   * Business Strategy
   * Creative Work

---

## Analysis Process

After collecting answers, the AI performs a structured evaluation:

1. Analyze user profile
2. Analyze primary use cases
3. Identify best Claude model
4. Determine when to use:

   * Haiku
   * Sonnet
   * Opus
5. Recommend optimal effort settings
6. Suggest situations for High and Max effort
7. Build a personalized workflow
